# WaifuX 相比 PlumWallPaper 多出的功能清单

> 请在你想要的功能前打勾 `[x]`，不需要的保持 `[ ]`，然后告诉我。

---

## 壁纸引擎

- [x] **Wallpaper Engine 兼容** — 支持 Steam Workshop 的 Scene/Web 类型壁纸，通过外部 CLI 桥接渲染
- [x] **Workshop 搜索与下载** — 直接在应用内搜索、预览、下载 Steam 创意工坊壁纸
- [x] **循环预处理（首尾 crossfade）** — 视频壁纸首尾 1 秒做淡入淡出过渡，消除循环播放的跳帧感
- [x] **超分辨率增强** — 对低分辨率壁纸进行 AI 超分放大
- [x] **原始壁纸保存/恢复** — 退出应用时自动恢复用户之前的系统桌面壁纸

## 在线壁纸源

- [x] **Wallhaven API 集成** — 接入 Wallhaven 在线壁纸库，支持搜索、分类、NSFW 过滤
- [x] **4K Wallpapers 备用源** — 第二在线壁纸源，自动切换
- [x] **可扩展规则引擎** — XPath 解析的数据源规则，支持用户自定义添加壁纸网站
- [x] **规则远程同步** — 数据源规则从 GitHub 仓库远程更新，无需发版

## 轮播调度增强

- [x] **每屏独立轮播配置** — 每个显示器可单独设置轮播开关、间隔、顺序
- [x] **静态壁纸轮播** — 轮播支持静态图片（当前 Plum 轮播仅视频/HEIC）
- [x] **Wallpaper Engine Scene 轮播** — 轮播支持 WE 场景类型

## 动漫/媒体生态

- [ ] **动漫追番系统** — 动漫搜索、详情、在线播放、观看进度追踪
- [ ] **弹幕系统** — 滚动/顶部/底部三种模式，15 轨道管理，可配置速度和透明度
- [ ] **Bangumi API 集成** — 接入 Bangumi 动漫数据库
- [ ] **媒体浏览器** — 多数据源的图片/视频浏览和下载

## UI / 设计系统

- [] **Liquid Glass 设计语言** — 适配 macOS 26+ 原生毛玻璃效果，低版本 fallback
- [] **纯 SwiftUI 重写 UI** — 从 WebView+React 迁移到原生 SwiftUI（大工程）
- [] **自定义窗口控件** — 自绘红绿灯按钮替代系统默认

## 下载管理

- [x] **下载任务管理器** — 支持暂停/恢复/取消/重试，进度持久化
- [x] **并发下载控制** — Actor 隔离的并发安全下载队列

## 系统集成

- [x] **国际化（i18n）** — 支持中文/英文/日文三语切换
- [x] **macOS 26 深度适配** — 处理 CFXPreferences 栈溢出、窗口动画崩溃等系统级 bug
- [x] **锁屏暂停 + 预览图** — 锁屏时暂停视频并显示静态预览图，解锁后恢复
- [x] **桌面 Space 同步** — 跨多个虚拟桌面同步壁纸状态
- [x] **应用内更新检测** — 检查 GitHub Release 新版本，自动下载安装
