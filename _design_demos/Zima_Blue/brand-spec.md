# Plum x Zima Blue · Brand Spec
> 灵感来源：Love, Death & Robots S1E14 "Zima Blue"
> 视觉目标：将 Plum 从工具软件升华为一种“追求视觉真理”的交互叙事。

## 🎯 核心资产 (Core Assets)

### 标志性符号 (Signatures)
- **The Blue Tile**: 齐马蓝正方形色块 (#36a9e1)。
- **The Pool Grid**: 泳池底部的网格纹理（作为背景或容器遮罩）。
- **LDR Icons**: ❤️, ❌, 🤖 (作为软件的版本/信息水印)。

### 气质关键词 (Vibe Keywords)
- **极简 (Minimalist)**: 每一个元素都必须赚取其存在感。
- **秩序 (Ordered)**: 严谨的几何比例，完美的 4:3 或 1:1 预览。
- **深邃 (Profound)**: 大量的留白（深空黑）与高对比度的蓝。

## 🎨 视觉系统 (Visual System)

### 色板 (Palette)
- **Zima Blue**: `#36a9e1` (主交互、焦点、生命力)。
- **Void Black**: `#080808` (全局背景、极致深度)。
- **Pool Grey**: `#2a2a2a` (边框、次要 UI、网格线)。
- **Mute Ink**: `#666666` (不活跃状态、辅助文本)。

### 字型 (Typography)
- **Display**: `Newsreader` (或类似的优雅衬线体，用于引语/叙事)。
- **Functional**: `-apple-system, system-ui` (极细/Medium，大字符间距)。
- **Mono**: `SF Mono` (用于技术参数、坐标)。

### 动效语法 (Motion)
- **Geometric Unfold**: UI 展开时应像机械板件折叠。
- **Exponential Easing**: `cubic-bezier(0.19, 1, 0.22, 1)` (极致的丝滑感)。
- **Glitch Fade**: 切换壁纸时伴随极细微的水平信号扫描动效。

## 🛠️ 交互逻辑 (Interaction)
- **无 UI 态**: 默认情况下，屏幕完全属于壁纸。
- **触发感应**: 鼠标进入边缘 20px 时，UI 像从泳池底部浮现。
- **齐马蓝验证**: 核心操作（如“应用壁纸”）完成后，屏幕中心闪烁一次齐马蓝正方形。
