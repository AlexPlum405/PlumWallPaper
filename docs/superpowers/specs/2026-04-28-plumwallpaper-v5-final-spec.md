#  PlumWallPaper v5 全功能技术与交互规格说明书

**日期**：2026-04-28  
**版本**：v5.2.4 (Pro Edition)  
**目标**：定义 macOS 平台顶级离线动态壁纸管理器的交互与技术基准。

---

## 一、 全局组件与视觉规范

### 1.1 顶部导航栏 (TopNav)
*   **Branding (左侧)**：
    *   **Logo**：包含 Apple Metal 风格的渐变 Icon。
    *   **文字**：`Plum` (Cormorant Garamond 斜体, 32px) + `WallPaper` (Inter 细体, 10px, 字间距 0.45em)。
    *   **行为**：点击任何位置强制返回 `HomePage`。
*   **视图切换 (中间)**：
    *   **首页/壁纸库**：Pill 型切换器。非活动状态透明度 40%，活动状态增加 `rgba(255,255,255,0.08)` 背景及毛玻璃效果。
*   **工具组 (右侧)**：
    *   **搜索按钮**：圆形 `var(--glass)` 按钮，触发沉浸式搜索遮罩。
    *   **添加按钮**：触发 `ImportModal`。
    *   **设置按钮**：进入 `SettingsPage`。

### 1.2 全局动效
*   **入场动画**：`fadeIn` (10px 位移) + `cubic-bezier(0.2, 1, 0.3, 1)`，时长 `0.6s`。
*   **悬停反馈**：统一采用背景变亮 + `translateY(-1px)` 的轻微浮动感。

## 二、 首页 (HomePage) 规格

### 2.1 Hero Section (顶部沉浸区)
*   **预览层**：支持 `object-fit` 适配模式联动。底部覆盖 `#0d0e12` 向透明的线性渐变（遮罩高度约 50%）。
*   **信息层**：
    *   **Tag**：大写间距，Plum Red 高亮。
    *   **Title**：`Cormorant Garamond` (84px)，负字间距。
    *   **Metadata**：Type/Res/Size/Duration 四维展示。
*   **操作组**：
    *   **设为壁纸**：主动作按钮（White Background），点击触发 `MonitorSelector`。
    *   **收藏**：次动作按钮，图标由 `Icon name="heart"` 构成。
*   **Thumb Strip (缩略图条)**：
    *   **交互**：支持触控板横向惯性滚动，支持左键长按拖拽（逻辑需区分 Drag 和 Click，位移超过 5px 判定为拖拽，不触发切换）。
    *   **状态**：当前项 100% 不透明 + 白色边框；非当前项 40% 不透明。

### 2.2 沉浸预览区 (PreviewPage)
*   **视觉净化**：彻底移除 macOS 红绿灯按钮，取消“窗口”暗示，画面完全回归壁纸本身。
*   **顶部布局**：采用居中的 `Immersive Preview` 胶囊标签，右上角保留一个带毛玻璃效果的 `Close` 按钮。

### 3.2 调节面板 (ColorPage)
*   **界面净化**：同样移除红绿灯，通过左上角的悬浮 `Close` 按钮返回。这种“全屏画布”式的交互模拟了专业调色软件（如 Davinci 或 Lightroom）的沉浸感。

### 2.3 右键菜单 (ContextMenu)
*   **选项列表**：
    1. **设为壁纸**：直接进入显示器分配。
    2. **色彩调节**：进入 `ColorPage`。
    3. **收藏**：UI 状态切换。
    4. **删除**：Plum Red 警示色，触发二次确认逻辑。

## 三、 专业色彩调节页 (ColorPage)

### 3.1 预览容器
*   **滤镜链 (Filter String)**：
    `hue-rotate` + `saturate` + `brightness` + `contrast` + `blur` + `grayscale` + `invert`。
*   **暗角层 (Vignette)**：
    使用 `radial-gradient` 模拟，中心透明，边缘根据滑块值向黑色过渡。
*   **颗粒层 (Grain)**：
    基于 SVG `feTurbulence` 分形噪声滤镜的 `data:image` 动态填充，通过不透明度控制强度。

### 3.2 调节面板 (360px Width)
*   **基础校正组**：曝光度 (0-200%)、对比度、饱和度、色调。
*   **艺术效果组**：模糊 (0-20px)、颗粒感 (0-100)、暗角 (0-100)。
*   **风格转换组**：灰度 (0-100%)、反转 (0-100%)。
*   **预设系统**：
    *   **Pills**：提供“胶片”、“深夜”等一键滤镜预设。
    *   **保存当前**：弹出 Modal 输入名称。

## 四、 偏好设置 (SettingsPage Pro Edition)

### 4.1 架构与对齐规范
*   **双栏布局**：采用 `Sidebar (240px) + Content (Fluid)` 的工业级架构。
*   **垂直基准线对齐 (Vertical Baseline)**：
    *   **规范**：侧边栏第一个导航项的文字顶端必须与内容区大标题文字顶端处于同一物理水平线。
    *   **实现**：统一通过 `padding-top: 140px` 锚定起始位置，消除视觉上的错位感。
*   **去盒子化设计**：主内容区移除多余的圆角容器边框，采用流式全高布局，利用 `60px` 深度模糊背景营造沉浸感。

### 4.2 材质渲染策略 (Bug 规避)
*   **侧边栏 (Sidebar)**：
    *   **背景**：采用稳健的实体色 `#14151a`，彻底弃用 `backdrop-filter`。
    *   **目的**：物理性解决 WebKit 引擎在处理复杂嵌套布局时产生的“垂直分裂伪影”。
    *   **边界**：右侧使用 `1px` 极细白边 (`rgba(255,255,255,0.05)`) 替代外阴影。
*   **内容区 (Content)**：
    *   **背景**：`rgba(255,255,255,0.01)` + `80px` 深度毛玻璃，呈现极致通透度。

### 4.3 性能监控仪表盘 (Real-time Vitals)
*   **动态波形 (Simulated Waves)**：
    *   **指标**：GPU 渲染压力、动态帧率 (FPS)、系统内存占用。
    *   **动效**：每个指标下方配备 24 条动态变化的柱状图，模拟系统实时负载波动。
*   **显示规格**：采用 Monospaced 字体增强“仪表感”，辅以 Accent 色彩高亮关键数值。

### 4.4 智能暂停策略 (Smart Pause - 全量 9 项)
为确保系统级效能，必须全量支持以下场景的自动避让：
1.  **电池模式**：断开电源时自动停止渲染。
2.  **全屏应用**：当检测到游戏或工作软件占据整个屏幕。
3.  **遮挡感知**：当所有桌面窗口覆盖了壁纸表面。
4.  **低电量状态**：配合 macOS 低电量模式强制休眠。
5.  **屏幕共享/录制**：保护隐私并确保演示流程顺畅。
6.  **笔记本闭盖模式 (Clamshell)**：仅在外接显示器需要时运行。
7.  **高负载软件避让**：检测到 Final Cut Pro, Blender 等专业软件开启。
8.  **失去焦点 (Inactive)**：当前活跃窗口不是桌面时自动暂停。
9.  **系统预休眠**：在进入待机状态前提前停止渲染引擎。

### 4.5 物理拓扑编辑器 (Topology Editor)
*   **高保真模型**：使用带 16:10 比例的物理显示器卡片进行排列。
*   **色彩协议标记**：在物理模型右上角展示 P3 / sRGB 协议标签。
*   **交互逻辑**：支持模拟拖拽物理位置（视觉反馈），主显示器下方增加 Accent 色下划线标记。

## 五、 导入中心 (ImportModal)

*   **拖拽区**：
    *   **动态边框**：悬停时边框由 `rgba(255,255,255,0.1)` 变为 `Plum Red`。
    *   **格式 Pills**：展示 Video, 8K+, HEIC, ProRAW 标签。
*   **导入选项**：
    *   **批量文件**：触发系统文件选择器。
    *   **整包导入**：触发 `webkitdirectory` 文件夹扫描。
*   **分析状态**：进度条配合“请勿关闭应用”的风险提示。
