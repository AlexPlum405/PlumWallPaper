# PlumWallPaper 完整功能测试清单

## 前置准备

### 1. 添加 SwiftSoup 依赖
- [ ] 按照 `SWIFTSOUP_SETUP.md` 添加 SwiftSoup 包
- [ ] 验证 MediaService 能正常导入 SwiftSoup

### 2. 清理 Xcode 缓存
- [ ] Product → Clean Build Folder (Cmd+Shift+K)
- [ ] 删除 DerivedData: `rm -rf ~/Library/Developer/Xcode/DerivedData/PlumWallPaper-*`
- [ ] 重新编译项目

### 3. 初始化数据源管理器
在 `AppDelegate.swift` 或 `PlumWallPaperApp.swift` 的启动方法中添加：
```swift
// 恢复壁纸源状态
WallpaperSourceManager.shared.restoreState()

// 启动时选择数据源
Task {
    await WallpaperSourceManager.shared.performStartupSourceSelection()
}

// 启动网络监控
await NetworkMonitor.shared.startMonitoring()
```
